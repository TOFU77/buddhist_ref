# Buddhist Reference

仏教・世界古典の聖句辞典

🌐 **https://tofu77.github.io/buddhist_ref/**

---

## 概要

仏教経典・世界の古典聖典から厳選した約500句を、14カテゴリに分類して収録した参照アプリです。  
サーバー不要のスタンドアロンHTML（バニラJS）で動作します。

> 本アプリは比較研究・個人の思索を目的とした参照ツールであり、特定の宗教の優劣を論じるものではありません。

## 機能

- **カテゴリ別ブラウズ** — バブルをタップしてカテゴリ一覧へ
- **テキスト検索** — 原文・訳文・主題をリアルタイム検索
- **Wikipedia連携** — 各句からキーワード検索（PC: 右半分パネル、モバイル: フルスクリーン）
- **足跡** — 閲覧履歴をlocalStorageに保存、右スワイプで表示
- **ダークテーマ** 固定

---

## 技術仕様

- バニラHTML / CSS / JavaScript（フレームワーク不使用）
- 外部依存なし（Wikipedia REST APIのみ使用）
- `index.html` + `db_81.js` 〜 `db_94.js` の15ファイル構成
- GitHub Pages で静的ホスティング
